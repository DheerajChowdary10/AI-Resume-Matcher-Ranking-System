AI Resume Matcher & Ranking System

An AI-powered Resume Compatibility Analyzer that evaluates resume alignment with job descriptions using Natural Language Processing (NLP) and Machine Learning techniques.

 Overview

The AI Resume Matcher & Ranking System is designed to automate resume screening by computing semantic similarity and keyword matching between resumes and job descriptions.

The system generates a compatibility score and helps rank candidates based on job relevance.

 Features

Upload Resume in PDF format

Paste Job Description

Automated Text Extraction

NLP-based Text Preprocessing

TF-IDF Vectorization

Cosine Similarity Computation

Keyword Frequency Matching

Final Compatibility Score Generation

Resume Ranking Capability

Interactive Streamlit Interface

Machine Learning Concepts Used

Natural Language Processing (NLP)

Text Normalization

TF-IDF Feature Extraction

Cosine Similarity

Unsupervised Similarity Modeling

Keyword-Based Scoring
